# scoop-bucket-jover

- [naotu](https://github.com/NaoTu/DesktopNaotu) - 脑图桌面版
- [pcmaster](https://mofang.ruanmei.com/) - 软媒魔方

- oracle-instance-client - Oracle 客户端库
